# ToDoList

ToDoList - список, с помощью которого можно управлять задачами:
- добавлять
- отмечать (не)завершенные задачи
- менять местами
- удалять (в режиме item и свайпом влево)

- бейдж на иконке приложения (количество невыполненных дел)
